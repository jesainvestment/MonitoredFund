## GitHub Pages Deployment
1. Create a GitHub repository.
2. Upload all files.
3. Go to Settings → Pages.
4. Select:
   - Source: Deploy from branch
   - Branch: main
   - Folder: /(root)
5. Save.

Your dashboard will be live in a few minutes.
